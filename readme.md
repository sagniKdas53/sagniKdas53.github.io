# A personal page

## Technologies used

1. Node Js
1. Sass
1. Bootstarp

## Setup to create the webpack

1.     npm install
2.     npx get-google-fonts -i "https://fonts.googleapis.com/css2?family=Open+Sans"
3.     npm run compile
4.     npm run build
